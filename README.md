## Características

- **Crear tareas pendientes**: Añade nuevas tareas con títulos y descripciones.
- **Editar tareas**: Modifica tareas existentes.
- **Eliminar tareas**: Elimina tareas de tu lista cuando ya no las necesites.
- **Marcar como completada**: Cambia fácilmente el estado de finalización de las tareas.
- **Ver tareas completadas**: Vista independiente para las tareas completadas.
- **Restaurar tareas**: Restaura las tareas completadas a la lista principal.

## Tecnologías utilizadas

- Angular
- TypeScript
- Angular Material for UI components
- Local Storage for data persistence

## Servidor de instalación y desarrollo

1. Instalar las dependencias:
```bash
npm install
```
2. Iniciar el servidor de desarrollo:
```bash
npm start
```

Vaya a `http://localhost:4200/` en su navegador para ver la aplicación. La aplicación se recargará automáticamente si modificas alguno de los archivos fuente.

> **Nota:** Asegúrate de tener una versión actualizada de Node.js instalada en tu equipo.
   **Nota:** para borrar LocalStorage agregar siguiente linea en .ts   localStorage.clear();
